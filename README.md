# 🍔 Online Food Ordering System

A full-stack web application that allows users to browse food items, add them to cart, and place orders seamlessly.

## 🚀 Project Type
Capstone Project (Wipro Pre-Skilling / Pre-Onboarding Training)

## 🛠️ Tech Stack
- Frontend: React JS
- Backend: Spring Boot (REST API)
- Database: MySQL
- Tools: Git, GitHub, Postman

## ✨ Features
- 🔐 User Registration & Login
- 🍽️ Browse Food Items
- 🛒 Add to Cart & Remove Items
- 💳 Payment Integration (UI-based)
- 📦 Order Placement & Tracking
- 📊 Admin Panel (optional if you added)

## 📂 Project Structure
- frontend/ → React Application
- backend/ → Spring Boot APIs
- database/ → SQL Scripts

## 🧠 Key Learnings
- Full Stack Development (React + Spring Boot)
- REST API Integration
- State Management in React
- Database Design & CRUD Operations
- Deployment & Version Control
